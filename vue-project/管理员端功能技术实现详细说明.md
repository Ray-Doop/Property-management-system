# 管理员端功能技术实现详细说明

## 一、登录模块

### 1.1 登录实现
- 前端使用Vue 3 Composition API + Element Plus表单组件
- 调用`/LoginRegister/adminlogin`接口，携带用户名、密码、验证码信息
- 后端Spring Security认证，生成JWT Token（含username、adminId、role），存入Redis（Key: `login:admin:{adminId}`）
- 前端存储token到localStorage，根据role跳转对应页面

### 1.2 路由守卫
- Vue Router `beforeEach`守卫，验证role权限（ADMIN/SUPER_ADMIN/OPERATOR）
- 未登录或权限不足时重定向到登录页

## 二、个人信息模块

### 2.1 信息获取与修改
- 调用`/LoginRegister/GetAdminData`接口，支持Redis缓存（20分钟过期）
- 头像上传使用Element Plus `el-upload`组件，调用文件上传接口
- 退出登录调用`/LoginRegister/logout`，清除Redis缓存和localStorage

## 三、数据可视化模块

### 3.1 数据统计概览
- 调用`/admin/stats/overview`接口，返回用户数、管理员数、帖子数等统计数据
- 使用ECharts绘制折线图，展示登录人数、发帖数、评论数、出行码使用趋势

## 四、公告管理模块

### 4.1 公告列表与发布
- 调用`/notice/List`接口，支持分页、标题搜索、状态筛选
- 富文本编辑器（wangeditor），附件上传至`/files/upload`
- 发布/保存草稿调用`/notice/publish`或`/notice/saveDraft`接口

### 4.2 公告操作
- 详情查看：`/notice/{noticeId}`
- 删除：逻辑删除（status=3），调用DELETE `/notice/{id}`

## 五、出行码管理模块

### 5.1 出行码列表
- 调用`/travel-pass/selectPage`接口，支持状态（ISSUED/ENTERED/EXITED）和用户名筛选
- 列表展示用户信息、通行类型、有效期、状态

## 六、报修管理模块

### 6.1 工单管理
- 调用报修工单查询接口，支持状态（待处理/已指派/维修中/已完成/已取消）和优先级筛选
- 分配维修人员：调用分配接口，更新status为"已指派"
- 取消工单：调用`/repair/cancel/{orderId}`，更新status为"已取消"

## 七、论坛管理模块

### 7.1 帖子管理
- 调用`/Forum/SelectPage`接口，支持分页、标题搜索
- 置顶/加精：更新status字段（3-置顶/2-加精）
- 删除帖子：逻辑删除，调用DELETE `/Forum/delMyPost`

### 7.2 评论管理
- 支持删除评论、禁言用户操作

## 八、物业缴费管理模块

### 8.1 账单管理
- 调用`/api/fee/allBills`接口，支持按residenceId或月份筛选
- 发布月度账单：调用`/api/fee/publishMonthly`，批量生成账单（amount=area*unitPrice）
- 账单状态：0-已取消/1-待支付/2-已支付

## 九、出行码扫描验证

### 9.1 扫码验证
- 上传二维码图片至`/travel-pass/verify`接口
- 后端`QRCodeUtil.decodeQRCode()`解析，验证record有效性和过期时间
- 根据status更新通行状态（ISSUED→ENTERED→EXITED），计算停车费
